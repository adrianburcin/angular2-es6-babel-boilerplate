# Start building Angular 2 apps with ES6(Babel) and Gulp
# Angular 2 ES6(Babel) Boilerplate

## Try

###### Install
```
npm install
```

###### Run
```
npm start
```
Open your favourite browser and navigate to localhost:3001.
